# Logical Reasoning - Races MCQs

### Q1.
**Problem:** In a 100m race, A can beat B by 25m and B can beat C by 4m. In the same race, A can beat C by:
A) 28m
B) 29m
C) 30m
D) 21m

**Answer:** A
**Explanation:**
1. When A covers 100m, B covers (100 - 25) = 75m.
2. When B covers 100m, C covers (100 - 4) = 96m.
3. Ratio of A:B = 100:75 = 4:3.
4. Ratio of B:C = 100:96 = 25:24.
5. Ratio of A:C = (A/B) * (B/C) = (4/3) * (25/24) = 100 / 72.
6. So, when A covers 100m, C covers 72m.
7. Distance by which A beats C = 100 - 72 = 28m.

### Q2.
**Problem:** In a 500m race, the ratio of the speeds of two contestants A and B is 3 : 4. A has a start of 140m. Then, A wins by:
A) 20m
B) B wins by 20m
C) A wins by 40m
D) B wins by 40m

**Answer:** B
**Explanation:**
1. To finish the race, A has to cover (500 - 140) = 360m.
2. B has to cover 500m.
3. Ratio of speeds A:B = 3:4.
4. Time taken by A to cover 360m = 360/3 = 120 units of time.
5. In 120 units of time, B covers = 120 * 4 = 480m.
6. When A finishes the race, B is at 480m (still 20m behind the finish line).
7. Wait, A finishes 360m in 120 units. In that time B covers 480m. B is at 480m when A finishes. So A wins?
8. Let's re-calculate for B.
9. Time taken by B to cover 500m = 500/4 = 125 units of time.
10. In 125 units, A covers = 125 * 3 = 375m.
11. Since A needs only 360m to win, A finishes first.
12. When A covers 360m (Time = 120), B covers 480m.
13. So A wins by (500 - 480) = 20m.

### Q3.
**Problem:** In a 100m race, A beats B by 10m and C by 13m. In a race of 180m, B will beat C by:
A) 6m
B) 5m
C) 4.5m
D) 4m

**Answer:** A
**Explanation:**
1. A:B = 100:90.
2. A:C = 100:87.
3. B/C = (B/A) * (A/C) = (90/100) * (100/87) = 90/87.
4. When B covers 90m, C covers 87m.
5. In a 180m race, when B covers 180m, C covers (87/90) * 180 = 87 * 2 = 174m.
6. B beats C by 180 - 174 = 6m.

### Q24.

8. L^2 - 18L = L^2 - 20L + 96.
9. 2L = 96 => L = 48m.
