### Q1.
**Problem:** Find the missing letters: SCD, TEF, UGH, ____, WKL
A) CMN
B) UJI
C) VIJ
D) IJT

**Answer:** C
**Explanation:**
1. First letter: S -> T -> U -> V -> W (Consistently +1)
2. Second letter: C -> E -> G -> I -> K (Consistently +2)
3. Third letter: D -> F -> H -> J -> L (Consistently +2)
So, the missing group is VIJ.
